# Numeto

**Status:** pre-engagement. Numeto is the current employer, framed as the first
retained client of Neutral Working. The long-term direction is a transition from
employee to Fractional PO for the same company, but no formal client–vendor
relationship exists yet.

## What this means in practice

- **Operationally, nothing changes.** Day-to-day work is still employer work
  under the existing employment relationship.
- **The Numeto-as-client framing is internal.** Useful for shaping what a
  Neutral Working Fractional PO offer looks like in practice; not yet a
  declaration to anyone outside this repo.
- **Employer IP stays employer IP.** Anything specific (ticket IDs, product
  names not already in public docs, internal roadmaps, customer data, PRDs)
  does not live in this directory or in any public surface of the repo.

## The engagement, as if it were one

The shape this is being run as:

| Piece | Cadence | Artefact |
|---|---|---|
| Morning triage | Weekday mornings, ~45 min | Honest tracker state + drafted standup. See [writing/morning-triage.html](../../writing/morning-triage.html). |
| PRD authorship | As required | Long-form scope docs for in-flight features. |
| Stakeholder grilling | Embedded in the morning triage | Surfacing fake blockers, priority drift, status mismatches. |
| Weekly recap | Fridays | "Shipped this week / blocked / next week's priorities" — drafted from the tracker. |

## Where work lives

- **Tracker:** Airtable `Tasks` base (private to the client). Ticket-level rows,
  status vocabulary documented in `writing/morning-triage.html`.
- **PRDs:** in the tracker row's Notes field — the row is the canonical doc.
- **Standups:** delivered in the client's chat tooling.
- **This directory:** scope notes and engagement metadata only.

## When this becomes real

When any of the following happen, this README graduates from "imagined" to
"current":

- A signed agreement covering scope, cadence, and rate.
- A handover of the existing employer responsibilities to a successor or to
  the client team.
- A public-facing change (Neutral Working portfolio page lists Numeto as a
  client) — only after the above two.

Until then, treat this directory as preparation, not announcement.
